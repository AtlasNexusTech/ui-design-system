Pourquoi ce skill

La qualité visuelle n'est pas subjective

Les agents IA génèrent du code UI par défaut — couleurs génériques, espacements cassés, animations robotiques. Ce skill remplace les décisions arbitraires par des règles précises.

C'est un pack de compétences pour Claude Code — intégré dans le .claude/skills/ du projet. Une fois chargé, l'agent design avec la même exigence qu'un directeur artistique.

Architecture

5 modules, une exigence

Chaque module encode un réflexe design précis. Ensemble, ils couvrent tout le cycle — de l'audit initial à la validation en conditions réelles.

01

Audit

Cheap Detector

Scanner automatique de 9 signaux de médiocrité avant de livrer une UI. Gradients génériques, espacements cassés, animations robotiques, ombres Bootstrap — tout ce qui fait « cheap » est détecté et corrigé.

✕ purple→pink ✕ 0 2px 4px shadow ✕ border-radius: 8px ✕ #000 sur #fff ✕ transition: all 0.3s ✕ Inter 16px regular
02

UX

Règles Embarquées

10 règles UX fondamentales appliquées automatiquement. Hiérarchie, feedback, performance perçue, zones tactiles — l'agent ne génère plus d'erreurs UX de débutant.

Max 3 tailles typo 1 CTA primaire F-pattern 4 états interactifs Validation inline 44×44px tap zone 200ms transitions
03

Taste

Culture Visuelle

Règles typographiques, palettes par contexte, courbes d'animation. L'agent sait quelles combinaisons fonctionnent et lesquelles bannir. Inter, Roboto, Open Sans sont blacklistés comme polices principales.

Typographie ✓

Rubik + Nunito Sans

Playfair Display + Inter

DM Serif + DM Sans

Motion ✓

cubic-bezier(0.22,1,0.36,1)

stagger 40-80ms

spring au lieu de ease

Palettes

Finance → amber accent

SaaS → electric blue

Santé → vert sauge

04

Prototypage

Haute Fidélité

Extraction pixel, zéro interprétation libre, composants réutilisables, variables CSS systémiques. Ce qui est montré est reproduit.

const

spacing = { xs: 4, sm: 8, md: 16 };

// Zéro valeur en dur répétée

05

Validation

Playwright MCP

L'agent voit l'interface dans un vrai navigateur et valide le rendu. Screenshots, vérification des espacements, interactions. Plus jamais de surprise après déploiement.

browser_snapshot browser_take_screenshot browser_click
